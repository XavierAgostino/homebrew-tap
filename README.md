# homebrew-tap

Homebrew formulae for [@XavierAgostino](https://github.com/XavierAgostino)'s tools.

```bash
brew install XavierAgostino/tap/macstrap
```

Formulae in `Formula/` are published automatically by
[GoReleaser](https://goreleaser.com) when
[macstrap](https://github.com/XavierAgostino/macstrap) cuts a release —
don't edit them by hand.
